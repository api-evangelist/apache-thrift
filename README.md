# Apache Thrift (apache-thrift)
Apache Thrift is a software framework for scalable cross-language services development. It provides a code generation engine and IDL for building RPC services that work efficiently across C++, Java, Python, PHP, Ruby, Erlang, Go, JavaScript, and many more languages.

## APIs

### Apache Thrift
Thrift provides an Interface Definition Language (IDL) and code generation engine for building cross-language RPC services, with transport and protocol APIs supporting binary, compact, JSON protocols over HTTP, TCP, and other transports in 20+ languages.
